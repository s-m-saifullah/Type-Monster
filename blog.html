Both localStorage and sessionStorage are browser based data storing mechanism. Even though they work allmost similarly, there is one key difference between them.
localStorage data has no expiration date. They are available even if the browser is closed. On the other hand, sessionStoage has expiration time. They will be lost as soon as the tab or browser is closed.

Scope is a feature of variables. There are two types of scope: global scope and block scope or local scope.
If a variable is declared in the global scope that is called a global variable. A global variable can be defined anywhere in the JavaScript code and can be accessed from anywhere. global variable can be declared with var keyword
If a variable is declared in the block/ local scope that is called a local variable. A local variable can only be visible within two braces: function, if block, else block etc. Local variables can be declared with let or const keyword.
